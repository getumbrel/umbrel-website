Frequently Asked Questions

How can I uninstall the Bitcoin and Lightning node?

Currently, Umbrel installs a Bitcoin and Lightning node by default and it is not possible to remove it. Over the coming weeks, we‚Äôll migrate the Bitcoin and Lightning node to the Umbrel App Store and your Umbrel would then start from a clean slate. Learn more.

Do I need a display to plug into the Raspberry Pi?

No, a display is not needed. You can access your Umbrel on any device connected to the same network at http://umbrel.local in a web browser.

Can I use a 500GB drive instead of 1TB?

Because a Bitcoin node is currently installed by default on Umbrel, a 500GB drive is not enough to store the entire Bitcoin blockchain. However, we're working to migrate the Bitcoin and Lightning node to the Umbrel App Store, and once the migration is complete, you'll be able use a smaller drive depending upon your use case.

What are the credentials to SSH into my Umbrel?

ssh umbrel@umbrel.local (or ssh umbrel@<IP¬†address of your Umbrel>). The password is the same as your Umbrel's password. If you haven't set a password yet, the default password is:¬†moneyprintergobrrr.

Can I use a Raspberry Pi 3 to run Umbrel?

Unfortunately not. The Raspberry Pi 3 is not powerful enough to run Umbrel and the apps on it.

How do updates work on Umbrel?

Everytime a new Umbrel update is released, you'll see an alert in your Umbrel's dashboard. Just click "Install now"¬†to install the update.

How can I build or package an existing app for Umbrel?

Building or packaging existing apps for Umbrel is super simple. Please refer to our one-page guide here.

Can I plug two drives in the Raspberry Pi for extra storage?

Unfortunately not. The Raspberry Pi can run into power issues when connecting two drives simultaneously.
